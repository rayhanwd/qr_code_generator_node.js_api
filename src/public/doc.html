Introduction

A QR code generator is a web application that allows users to create a QR code from a text or URL input. The generated QR code can be used for various purposes, such as to store a website URL, phone number, or any other data.

How it work

This documentation outlines the implementation of a QR code generator web page using HTML, CSS, and a QR code API. The QR code API will be used to generate the QR code image in response to a request with the text to encode as the query parameter. The web page will have a form where the user can input the text, and upon submission, the QR code image will be displayed on the page.

Example request

http://localhost:3000/qrcode?text=type your text here
